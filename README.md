## Food Delivery Platform
### Introduction
This repository contains a Java-based food delivery platform developed with Java Swing and modeled by the Uber Eats application. It follows an object-oriented programming approach to build an interactive graphical user interface and manage user interactions.<br><br>
The platform integrates a database to store and manage customer information and order records. It demonstrates GUI development, object-oriented design, database integration, and the implementation of a complete food ordering workflow.
